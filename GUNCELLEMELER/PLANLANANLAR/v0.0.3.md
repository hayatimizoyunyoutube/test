# v0.0.3 — RAWG Kapak ve Meta Otomasyonu

## Plan

- Admin oyun formuna RAWG oyun arama kutusu eklenecek.
- Oyun adına göre RAWG arama yapılacak.
- RAWG sonucundan kapak görseli, arka plan görseli, tür, platform, çıkış tarihi, puan ve RAWG ID/slug bilgileri forma otomatik aktarılacak.
- RAWG sonuçları kartlı profesyonel arayüzle gösterilecek.
- RAWG API hataları admin panelde anlaşılır uyarı olarak gösterilecek.
