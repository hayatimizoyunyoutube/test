# v0.0.4 — YouTube Playlist Otomasyonu

## Plan

- Admin oyun formuna YouTube playlist arama/kontrol alanı eklenecek.
- Playlist URL girilince playlist ID otomatik çözülecek.
- YouTube Data API ile playlist videoları çekilecek.
- Videolar `game_episodes` tablosuna bölüm olarak kaydedilecek.
- Bölüm sayısı ve yayın ilerlemesi otomatik güncellenecek.
- YouTube senkron sonucu admin panelde tablo halinde gösterilecek.

## Supabase Notu

v0.0.4 için mevcut `game_episodes` tablosu yeterli görünmektedir. Yeni alan gerekirse ayrıca belirtilecek.
