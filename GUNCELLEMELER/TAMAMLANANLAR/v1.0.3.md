## Eklenenler

- Ana sayfa referans görsele daha yakın profesyonel yapıya taşındı
- Kompakt üst header eklendi
- Büyük sinematik hero/banner alanı eklendi
- Arşiv Hazır / Hayatımız Oyun tarzı güçlü vitrin hazırlandı
- Devam Eden Seriler bölümü eklendi
- Son Eklenen Videolar bölümü eklendi
- Kesintisiz Deneyim / İlerlemeyi Kaydet / Listeler / Topluluk kartları eklendi
- Tamamlanan / Devam Eden / Yakında Gelecek hızlı arşiv alanı eklendi
- Kırmızı, mor ve mavi vurgu renkleri güçlendirildi
- Mobil ana sayfa görünümü iyileştirildi
- Public iç sayfalara ortak üst menü eklendi
- Seriler, kategoriler, kanallar, güncellemeler ve seri detay sayfalarında menü kaybolmaması sağlandı
- Normal site mantığında sayfalar arası geçiş kolaylaştırıldı
- Arayüz referans görsele daha yakın koyu oyun sitesi temasına taşındı
- Tüm public sayfalarda ortak profesyonel header güçlendirildi
- Sayfalar arası geçiş normal site mantığına yaklaştırıldı
- Seri, kategori, kanal ve güncelleme kartlarının görsel dili eşitlendi
- Kırmızı / mor / mavi vurgu renkleri genel siteye yayıldı
