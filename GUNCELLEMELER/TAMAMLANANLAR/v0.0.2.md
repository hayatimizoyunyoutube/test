# v0.0.2 — Admin Oyun Yönetimi

## Durum
Tamamlandı.

## Eklenenler

- `/admin/games` sayfası eklendi.
- Supabase `games` tablosundan admin oyun listeleme eklendi.
- Oyun ekleme formu eklendi.
- Oyun düzenleme formu eklendi.
- Oyun silme işlemi eklendi.
- Oyun adı, slug, açıklama, kapak URL, banner URL, tür, etiketler, platformlar, durum, çıkış tarihi, puan, bölüm ilerlemesi, RAWG bilgileri, YouTube playlist bilgileri, seri adı ve seri sırası alanları eklendi.
- Öne çıkarma ve yayında/gizli ayarları eklendi.
- Supabase env eksikse güvenli örnek veri gösterimi eklendi.
- Admin panel ana ekranındaki Oyun Yönetimi kartı aktif modüle bağlandı.
- Site ana sayfasındaki oyun kartları admin panelden eklenen public oyunlarla uyumlu hale getirildi.
- Supabase `schema.sql` v0.0.2 sürüm bilgisiyle güncellendi.
- Oyun listeleme performansı için temel indeksler eklendi.
- Bakım modu ayarı korunacak şekilde `maintenance_mode` insert işlemi yine `on conflict do nothing` olarak bırakıldı.

## Teknik Not

Admin kayıt işlemleri server action ile yapılır. `SUPABASE_SERVICE_ROLE_KEY` sadece server tarafında kullanılır, client tarafına gönderilmez.
