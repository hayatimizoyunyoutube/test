# Hayatımız Oyun - Sürüm Bilgisi

## Mevcut sürüm

v0.0.1

## Ana yayın hedefi

v4.0.0

## Proje tipi

YouTube oyun serileri arşiv video sitesi.

## Geliştirme kuralı

Bu proje küçük ve kontrollü sürümlerle geliştirilecek.

Her sürümde sadece belirli bir alan eklenecek.  
Her şeyi tek seferde eklemek yok.  
Önce public kullanıcıya görünen profesyonel arayüz yapılacak.  
Admin panel, yetkili yönetimi, Supabase Auth ve API entegrasyonları daha sonra eklenecek.

## v0.0.1 hedefi

- Temiz çalışan public ana sayfa
- 404 vermeyen deploy
- Profesyonel koyu tema başlangıcı
- Tamamlanan / Devam Eden / Yakında Gelecek Seriler alanları
- /api/health test endpoint

## v4.0.0 hedefi

v4.0.0 ana açılış sürümü olacak.

Bu sürüme kadar:

- Public arayüz tamamlanacak
- Seriler sistemi tamamlanacak
- Kategoriler tamamlanacak
- Kanallar tamamlanacak
- Supabase veri sistemi kurulacak
- YouTube playlist sistemi kurulacak
- Kayıt / giriş sistemi kurulacak
- Admin panel sadece yetkililere özel olacak
- Normal kullanıcıya admin alanları gösterilmeyecek
- Site profesyonel yayın seviyesine getirilecek
