# Changelog

## v0.0.3

- Admin oyun yönetimi aktif edildi.
- Oyun ekleme, düzenleme, silme ve listeleme eklendi.
- `/admin/games` sayfası oluşturuldu.
- Supabase service role sadece server tarafında kullanılacak şekilde CRUD akışı kuruldu.
- Oyun formuna RAWG ve YouTube için ileride kullanılacak alanlar eklendi.
- Supabase schema v0.0.3 sürüm bilgisi, indeksler ve güncelleme notuyla güncellendi.

## v0.0.1

- Next.js + Supabase + RAWG + YouTube altyapısı eklendi.
- Ana sayfa, admin başlangıç ekranı, kurulum dosyaları ve ilk Supabase şeması hazırlandı.
